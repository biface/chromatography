1//! Output module for simulation results
2//!
3//! This module provides tools to output simulation results in various formats:
4//! - **Visualization**: PNG/SVG plots using plotters
5//! - **Export**: CSV/JSON data export for external analysis
6//!
7//! # Architecture
8//!
9//! ```text
10//! output/
11//! ├── mod.rs              ← This file
12//! ├── visualization/      ← Plots and graphics
13//! │   ├── mod.rs
14//! │   └── steady.rs
15//! └── export/             ← Data export
16//!     ├── mod.rs
17//!     └── csv.rs
18//! ```
19//!
20//! # Quick Start
21//!
22//! ## Visualization
23//!
24//! ```rust
25//! use chrom_rs::output::visualization::{plot_chromatogram, PlotConfig};
26//! use chrom_rs::solver::SimulationResult;
27//! use chrom_rs::physics::{PhysicalState, PhysicalQuantity, PhysicalData};
28//!
29//! # let state = PhysicalState::new(PhysicalQuantity::Concentration, PhysicalData::Scalar(0.0));
30//! # let result = SimulationResult::new(vec![0.0, 1.0], vec![state.clone(), state.clone()], state);
31//! // Generate PNG plot
32//! let _ = plot_chromatogram(&result, 100, "/tmp/output.png", None);
33//! ```
34//!
35//! ## CSV Export
36//!
37//! ```rust
38//! use chrom_rs::output::export::{CsvExporter, CsvConfig, Exporter};
39//! use chrom_rs::solver::SimulationResult;
40//! use chrom_rs::physics::{PhysicalState, PhysicalQuantity, PhysicalData};
41//!
42//! # let state = PhysicalState::new(PhysicalQuantity::Concentration, PhysicalData::Scalar(0.0));
43//! # let result = SimulationResult::new(vec![0.0, 1.0], vec![state.clone(), state.clone()], state);
44//! let exporter = CsvExporter::default();
45//!
46//! // Export to CSV
47//! let _ = exporter.export_single(&result, None, "/tmp/data.csv");
48//! ```
49//!
50//! # Design Philosophy
51//!
52//! The output module separates concerns:
53//! - **Visualization**: For human interpretation (plots, graphs)
54//! - **Export**: For programmatic analysis (CSV, JSON, HDF5)
55//!
56//! Both submodules accept simple `&[f64]` slices for maximum flexibility.
57//!
58//! # Version History
59//!
60//! - **v0.1.0**: Static plots (PNG/SVG) and CSV export
61//! - **v0.2.0+**: Animations, HDF5, JSON export (planned)
62//!
63//! # Examples
64//!
65//! See `examples/` directory for complete workflows:
66//! - `examples/output_demo.rs` - Complete output pipeline
67//! - `examples/tfa_export.rs` - TFA chromatogram export
