Command-line interface.

Entry point is cli::build_app, which assembles the dynamic-cli application from the embedded commands.yml declaration and wires cli::app::RunHandler to the simulation pipeline. Command-line interface for chrom-rs.

Assembles the dynamic-cli application from a declarative YAML configuration embedded at compile time and wires it to the simulation pipeline defined in the app module.

§Entry point

chrom_rs::cli::build_app()
    .expect("CLI initialisation failed")
    .run();

§Command surface (v0.2.0)

chrom-rs run [--project-dir <dir>]

             [--output-csv   <file.csv>]
             [--output-plot  <file.png>]
             [--export-json  <file.json>]
