Builds the JSON export map for the simulation result.

Exportable::to_map is defined on concrete model types, not on Box<dyn PhysicalModel>. This helper re-reads and deserialises the model file into the appropriate concrete type to call to_map, keeping all concrete type knowledge confined to cli/app.rs.
