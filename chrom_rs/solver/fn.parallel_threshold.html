pub fn parallel_threshold() -> usize
Expand description

Return the current parallel-execution threshold.

PhysicalData::apply() uses sequential iteration when the data contains fewer elements than this value, and switches to Rayon when it contains more — but only when the crate is compiled with the parallel feature.

§Example

use chrom_rs::solver::parallel_threshold;

assert!(parallel_threshold() > 0);
