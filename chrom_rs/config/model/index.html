Loader for model.yml — returns a boxed PhysicalModel. Loader for model.yml / model.json.

model.yml describes the physical column and its parameters. The injection profile is intentionally absent (or set to None) here — it is supplied by scenario.yml and applied by scenario::load_scenario.

§File format

The file is deserialised via typetag, which uses the type name as the root key. Supported formats: .yml / .yaml (serde_yaml) and .json (serde_json). The extension determines the parser.

§LangmuirSingle example

LangmuirSingle:
  lambda: 1.2
  langmuir_k: 0.4
  port_number: 2.0
  porosity: 0.4
  velocity: 0.001
  column_length: 0.25
  nz: 100
  injection:
    type: None

§LangmuirMulti example

LangmuirMulti:
  species:
    - name: Ascorbic
      lambda: 1.0
      langmuir_k: 1.1
      port_number: 2
      injection:
        type: None
    - name: Erythorbic
      lambda: 1.0
      langmuir_k: 1.7
      port_number: 2
      injection:
        type: None
  n_points: 100
  porosity: 0.4
  velocity: 0.001
  column_length: 0.25

The injection: type: None placeholder is replaced by scenario::load_scenario after the model is loaded.
